Dolos

The spirit of trickery and guile. (Don't even ask me, I have no idea)

A browser extension for Chrome/Firefox using React + Material UI to transform various Anime database sites into fully fledged streaming sites with the power of the Grobber API.

Installing

You can head over to the latest release where you'll find a .crx file for Chrome and a .xpi file for Firefox which you may install.

Chrome Web Store

You may also download the Chrome extension from the Chrome web store here

Firefox Add-ons

I haven't gotten around to this yet, so please install the .xpi file from the release manually. The downside is, that the extension won't automatically update to newer versions.

Manual

If you wish to build and install the extension manually please refer to the "Getting ready for development" Guide

Documentation

There are some in-depth guides available on the Dolos website. If you want to dive straight in, here's a link to the API documentation.
